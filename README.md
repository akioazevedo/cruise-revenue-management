# Cruise Revenue Management & Pricing Optimization

This project simulates a revenue management system for cruise itineraries,
focusing on dynamic pricing, demand forecasting, and revenue optimization.

## Business Question
How should cruise cabin prices be adjusted over time to maximize revenue
given seasonality, inventory constraints, and competitive pricing?

## Tools
- Python (Pandas, Statsmodels)
- SQL (SQLite)
- Excel (pricing optimization)
- Tableau / Power BI
- AWS (optional)
